# ecommerce

Ecommerce is an application to know the price of the company products depending of the queried moment

Ecommerce has a service which you can read the price of the product

## Description SERVICE

#### URI

/ecommerce/api/productinfo

#### In-Parameters

**date:** Date to define the moment of the price. Format yyyy-mm-dd-HH.MM.SS

**productId:** Product identification number

**brandId:** Product brand identification number
